# Algorithm
在這邊會簡單介紹不同的演算法並利用Python實作，主要就是紀錄上課所學的內容

演算法就是將 input 透過一系列明確定義的計算步驟產生 output，input可能是空集合或是數值的集合，一個演算法的好壞取決於時間複雜度以及空間複雜度。

時間複雜度：在電腦上執行計算演算法的時間

空間複雜度：在電腦上執行計算演算法所佔用的記憶體資源
